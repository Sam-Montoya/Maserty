Doctype gets the version of the language of the page that it's written in. (DTD) It will also know what type of document it is.

Datatypes in databases can change the amount of space it takes up. You wouldn't want people being able to spam the database with tons of characters. This will make the load time of the database much much slower, due to the non-limited data type. Another example is if you wanted to store an object's ID in the database. You wouldn't want to have it be a string if the ID is all numbers! Or if a serial number contains letters and numbers, but is only 10 characters long, you could limit the amount of characters so space doesn't get wasted, along with long load times.
